# Introduction

Feelstatic is a static content management system (CMS) for websites using Next.js.

The concept is as simple as saving json files that can be accessed anywhere in your code.

Requirements:

- a GitHub account
- a Vercel account

You can get started by visiting the [Getting started](/documentation/getting-started) section. Below I will try to explain in more details what and how Feelstatic works.
